# Demo Script

## 30-second pitch

“TMT Bridge Lens v10 is a browser extension that makes English, Nepali and Tamang web content readable in-place. The judge chooses a source and destination language, clicks Translate page, and the extension preserves the original page while attaching sentence-level translations. For quick lookup, users can highlight any text and translate only that selection.”

## Live demo

1. Open the included `demo/test-page.html`.
2. Open the extension popup.
3. Set Source = English and Destination = Tamang.
4. Keep Bilingual mode selected.
5. Click Translate page.
6. Point out that original content is preserved.
7. Highlight one sentence.
8. Click Translate selection.
9. Switch Source/Destination and use Swap.
10. Switch to Replace mode.
11. Click Translate page again.
12. Click Restore page.

## Things to emphasize

- Simple source/destination UX.
- All required language pairs are present.
- Token is private and not hardcoded.
- Translation is sentence-level to match the API.
- Cache and deduplication make repeated demos feel instant.
- Restore makes the extension safe to use on real webpages.
